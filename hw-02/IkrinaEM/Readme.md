#ДЗ № 2

Определение тональности русского текста.
Скрипт, развернутый на приложении Streamlit производит оценку тональности введенного текста:
- нейтральный;
- позитивный;
- негативный.


Требуется предварительная установка библиотек (pip install transformers):
- streamlit
- torch
- transformers
- sentencepiece
